# Contributing

Thanks for your interest in contributing to zen!

## Getting started

1. Fork the repo and clone it
2. Run tests: `lua tests/run_all.lua`
3. Make your changes
4. Run tests again to make sure nothing broke
5. Open a pull request

## Guidelines

- Keep changes focused. One fix or feature per PR.
- Add tests for new features or bug fixes.
- Follow the existing code style.
- No new dependencies.

## Reporting bugs

Open an issue with a minimal reproduction. Include the Lua version you're using.

## Questions

Open an issue. There are no dumb questions.
